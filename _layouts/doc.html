Iniciando

Não iremos ensinar como instalar o WordPress pois essa é uma tarefa básica de todo desenvolvedor. Falaremos aqui do mínimo preciso para iniciar os seus trabalhos e ter uma ideia de como funciona o WP Starter Framework.

- Download

Baixe a última versão (0.9.0) do WP Starter clicando aqui ou clone o repositório git.

git clone https://github.com/mattdeveloper/wpstarter.git

- Instalação

Mude o nome da pasta wp-starter-framework para o nome do seu tema, por exemplo meu-tema. Jogue a pasta do tema para /wp-content/themes/ no local onde instalou o WordPress. Após feito a upload dos arquivos para o WordPress ative o tema em seu painel de administração.

Desenvolvedores

Quando o tema estiver ativo em seu WordPress será adicionado um novo link chamado Desenvolvedores no menu da esquerda do painel de administração. Na página que é aberta quando clicado o link você terá algumas opções com detalhes sobre elas para agilizar o desenvolvimento do seu tema e lhe poupar de escrever algumas linhas de codigo.

Caso queira esconder o link Desenvolvedores do menu do painel adicione a linha de código abaixo em qualquer lugar no arquivo functions.php do seu tema.

hide_developers();

Incluindo Arquivos

Para incluir arquivos css e javascript utilizando o wp_enqueue_scripts do WordPress pode ser um pouco confuso para alguns, por isso tentamos simplificar essa tarefa o máximo possível.

Recomendamos que utilize as funções do WP Starter Framework em vez de incluir os scripts direto no header.php ou no footer.php por ter mais controle e deixar o código HTML do seu tema mais “limpo”.

- CSS

Utilize as funções de css no arquivo functions.php do seu tema.

Para registrar novos css utilize a função new_css(‘nome-do-estilo’, ‘local-do-estilo.css’). Você precisa passar dois parâmetros nessa função sendo que o primeiro é o nome do estilo para quando for chamado e o segundo é o local onde está o arquivo. Para exibir o css no tema utilize a função use_css(‘nome-do-estilo’).

Normalmente para incluir arquivos no seu tema você precisa utilizar alguma função como por exemplo: get_bloginfo('template_directory') para que possa pegar o diretório do template, com a função new_css isso não é preciso, lhe poupando mais código. Abaixo daremos um exemplo de inclusão de css interno(dentro do tema) e externo(em outro lugar na internet).

<?php

// Arquivo CSS dentro da pasta CSS do tema.
// Não precisa incluir get_bloginfo('template-directory') na hora de especificar o caminho do css
new_css('page-home', 'css/page-home.css');
use_css('page-home');

// Arquivo CSS em outro servidor
// A função new_css irá reconhecer automaticamente que o arquivo é externo
new_css('font-raleway', 'http://fonts.googleapis.com/css?family=Raleway:400,200,500,700,800');
use_css('font-raleway');

?>

- CSS - Única Página

É recomendável que você carregue o arquivo css apenas nas páginas precise, no WP Starter Framework isso é feito de uma forma muito simples.

Quando for chamar usando a função use_css você pode passar dois parâmetros a mais tirando o nome do arquivo. O primeiro parâmetro como vimos anteriormente é o nome do arquivo, o segundo é a condição e o terceiro é o valor a ser validado.

Exemplo de uso:

<?php

// Abaixo incluímos o arquivo apenas quando a página é um template
// e quando esse template é o do arquivo contato.php
new_css('page-contato', 'css/page-contato.css');
use_css('page-contato', 'is_page_template', 'contato.php');

// Abaixo incluímos o arquivo apenas quando o usuário estiver acessando a home do site
new_css('page-home', 'css/page-home.css');
use_css('page-home', 'is_home');

?>

Condições aceitas no segundo parâmetro da função use_css e suas documentações.

  • is_home - Codex
  • is_front_page – Codex
  • is_single – Codex
  • is_singular – Codex
  • is_category – Codex
  • is_archive – Codex
  • is_page – Codex
  • is_page_template – Codex
  • is_search – Codex
  • is_admin – Codex

Opções do Tema

As Opções do Tema servem para que você edite textos, imagens, links, etc. do seu tema direto do painel do WordPress. Para criar um sistema de Opções de Tema no WordPress é preciso digitar bastante código, por este motivo o WP Starter Framework já vem com esse código integrado.

Desenvolvemos um código para as Opções do Tema e criamos algumas funções para que você não precise digitar nem uma linha de código, sendo assim, tudo será criado através da página Desenvolvedores que está localizada no menu do painel de administração do WordPress.
